# app-declaracion

https://hectorjaviermorenoh.github.io/app-declaracion/

Frontend para la gestión de declaraciones con React + Vite + React-Bootstrap.

## 🚀 Instalación
```sh
npm install
npm run dev
```

## 📦 Librerías incluidas
- React Router DOM
- React Bootstrap + Bootstrap
- Sass
- gh-pages
- sweetalert2


## 🌍 Deploy en GitHub Pages
```sh
npm run deploy
```
